## Setup

### 1. Get MetaMask
- Install from metamask.io
- Create wallet

### 2. Add BSC Testnet
In MetaMask:
- Network: BSC Testnet
- RPC: https://data-seed-prebsc-1-s1.binance.org:8545
- Chain ID: 97
- Symbol: tBNB
- Explorer: https://testnet.bscscan.com

### 3. Get Test BNB
- You need to have 0.002 BNB to receive the tBNB
- Go to https://testnet.bnbchain.org/faucet-smart
- Paste your address
- Wait a minute

### 4. Add Token42
- Click "Import tokens" in MetaMask
- Paste contract address: `0x47B1D3f793542fbc3D32e06DE9d3b376eeB4D921`
- Should auto-detect T42
